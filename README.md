# EvilNote — https://evilnote.su 

## What is EvilNote?

EvilNote is a professional email-sending tool. You can send emails to one person or to many at once (bulk), using your own SMTP servers. You get a simple dashboard where you manage everything: your sending accounts, recipient lists, email templates, and send history.

---

## What can you do?

- Send emails — One email or thousands. Choose your SMTP server, write the subject and body, pick your recipients, and send. You can also schedule sends for a specific date and time.
- Use your own SMTP — Add one or more SMTP accounts (Gmail, your hosting provider, or any SMTP server). You can rotate between several servers to spread the load.
- Manage recipient lists — Create lists of email addresses (paste them or upload a file). Use the same list for multiple campaigns. Lists support names so you can personalize emails (e.g. “Hi {{name}}”).
- Templates — Save subject and body as reusable templates. Use placeholders like {{email}} and {{name}} so each recipient gets a personalized message.
- Custom “From” — Set the sender address and name that recipients see (e.g. noreply@yourdomain.com or “Support Team”), and a Reply-To address if you want replies to go somewhere else.
- Logs & export — See the history of every send (who received it, subject, status). Export it to CSV for your records.
- Blacklist — Add addresses that should never receive your emails (bounces, opt-outs, test addresses). They are automatically skipped when you send to a list.
- Gifts — Receive codes or credentials (e.g. SMTP details) from the administrator in your Gifts tab. Copy and use them to add new sending accounts.
- Unsubscribe — If you add an unsubscribe link in your emails, recipients who use it are recorded and won’t receive future bulk sends from the platform.

---

## How do I start?

1. Sign up — Create an account with a password. Complete the security check (captcha) if asked. Save your login code (e.g. MN-XXXX-XXXX) — it is shown only once and you need it to log in.
2. Log in — Enter your login code and password (and captcha if shown). You’ll land in your dashboard.
3. Add an SMTP profile — In the SMTP tab, add at least one SMTP account (host, port, username, password). You can test it before sending.
4. Send — In the Send tab, choose your SMTP profile, add recipients (paste emails or select a list from Leads), write subject and body (or load a Template), optionally set a custom From/Reply-To, then click Send. For bulk sends you can set a delay between emails and use Schedule send to run at a chosen time.

For a step-by-step guide, use the Tutorial link from the dashboard or go to the Tutorial page after login.

---

## In short

EvilNote gives you a single place to send professional and bulk email: your SMTP accounts, your lists, your templates, and full control over sender identity and timing. Everything is in one dashboard, with logs and export so you stay in control.
